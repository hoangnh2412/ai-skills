## Quy ước

| Quy tắc | Mô tả |
|---------|--------|
| **Đặt tên file** | `YYYY-MM-DD-<stt>-<mo-ta-ngan>.md` |
| **Nội dung** | Tóm tắt trao đổi, phân tích, quyết định + lý do — không transcript word-by-word |
| **Nguồn gốc** | Trace về `assets/` khi trích dẫn tài liệu khách hàng |
| **Khi chốt** | Distill nội dung quan trọng → `docs/` (DOC tương ứng) |
| **Index phase** | Mỗi folder phase có `README.md` liệt kê các note |
