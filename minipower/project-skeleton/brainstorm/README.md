# Brainstorm — Trao đổi & phân tích

Phân tích, trao đổi, decision log — chốt nội dung → **distill** vào [`../docs/`](../docs/).

**Không chia folder con** — mỗi phiên ghi một file theo ngày.

## Đặt tên file

| Quy tắc | Ví dụ |
|---------|--------|
| `YYYY-MM-DD.md` | `2025-06-05.md` |
| `YYYY-MM-DD-<mo-ta-ngan>.md` | `2025-06-05-phan-tich-qa-discovery.md` |

Nhiều phiên trong cùng ngày → thêm hậu tố `-02`, `-03` hoặc mô tả ngắn.

## Nội dung gợi ý

- Tóm tắt trao đổi (không transcript word-by-word)
- Quyết định + lý do
- Phân tích từ `assets/` — trace nguồn khi cần
- Ghi phase Minipower đang làm (discovery, requirements, …) **trong file**
- Sau phiên: cập nhật tóm tắt vào [`../memory/{phase}/`](../memory/) — không nhồi vào `memory/memory.md`
